## Service Description

- SVG to TGS provides a browser-based workflow for uploading SVG artwork, configuring animation, previewing motion, and exporting Telegram-ready TGS files.
- Pro features may include expanded animation tools, Telegram preview, and Telegram pack integration where available.
- SVG to TGS is independent and is not an official Telegram product.

## User Responsibilities

- Use only artwork, logos, icons, fonts, and other assets that you have the right to use
- Do not use the service for illegal, abusive, misleading, infringing, or harmful content
- Check Telegram requirements before publishing stickers or custom emoji
- Keep backups of original SVG files and exported assets
