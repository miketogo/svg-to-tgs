# Privacy Policy

How SVG to TGS handles account data, analytics, support requests, and files used with Telegram preview or pack integration.

Canonical URL: https://svgtotgs.com/privacy
Markdown URL: https://svgtotgs.com/llms/privacy.md

## Overview

- The standard editor flow is browser-first: SVG import, animation setup, preview, and TGS export happen in the product interface.
- Connected features such as Telegram preview, pack actions, login, payments, analytics, and support communication require network requests.

## Information Processed

- Technical data such as browser type, device information, page URL, referrer, performance data, and interaction events
- Telegram login identifiers returned by Telegram authentication
- Payment and subscription status data needed to activate and manage Pro access
- Support messages sent to support@svgtotgs.com
- Generated TGS files or related metadata when Telegram preview or pack integration is used

## Contact

- Privacy questions and account requests: support@svgtotgs.com
