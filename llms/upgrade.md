# SVG to TGS Pro

PRO extends the SVG to TGS workflow with Telegram preview, sticker pack automation, custom emoji pack automation, and advanced animation controls.

Canonical URL: https://svgtotgs.com/upgrade
Markdown URL: https://svgtotgs.com/llms/upgrade.md

## Standard Plan

- Starter motion presets
- Basic export workflow
- Live browser preview
- Telegram-ready TGS export from SVG artwork

## PRO Features

- Preview generated TGS files in Telegram through the bot integration
- Add TGS animations to Telegram sticker packs
- Add TGS animations to Telegram custom emoji packs
- Full motion catalog, layer controls, timing controls, and intensity settings

## Payments

- Users log in before upgrading
- Available payment methods are loaded from the service configuration
- Supported configured methods can include Telegram Stars and crypto payments
